## Goal

Move from strong technical SEO and broad topical coverage to stronger rankings on the most valuable commercial and informational pages.

## What is already solved

- Sitemaps, robots, RSS
- Structured data
- Internal linking validation
- Duplicate image validation
- Static rendering on most important routes
- Asset compression and performance cleanup

## Upgrade checklist for each priority page

### Content depth
- Add more specific examples
- Remove any still-generic phrasing
- Add clearer user scenarios
- Add stronger comparisons

### Trust
- Add screenshots where useful
- Verify pricing and claims manually
- Add more “when not to use this” guidance
- Add more concrete drawbacks

### SERP intent
- Match the exact query better
- Tighten title and description if CTR is weak
- Improve first 150 words to answer intent faster

### Internal linking
- Add links from homepage or key hubs if the page matters commercially
- Add contextual links from related pages, not only related-card links
- Link up to the right hub and sideways to adjacent comparison/tutorial pages

## Search Console workflow

For priority pages:

1. Inspect URL
2. Request indexing
3. Monitor impressions and average position
4. If impressions rise but CTR is weak:
   - rewrite title
   - rewrite description
   - improve intro
5. If impressions stay low:
   - strengthen internal links
   - improve page uniqueness
   - add screenshots and richer examples

## What not to do next

- Do not add another large batch of generic articles before upgrading the best existing pages.
- Do not create more low-value template content before Search Console data confirms where traffic is already starting to appear.
- Do not chase “100 more pages” until the top 20 pages are materially stronger.
